✅ 1. Create a Cloudflare Account

Go to Cloudflare and sign up.
Verify your email.


✅ 2. Add Your Website

Click Add Site.
Enter your domain name (e.g., example.com).
Choose a plan (Free, Pro, Business, Enterprise).


✅ 3. DNS Configuration

Cloudflare will scan your existing DNS records.
Review and confirm the DNS records.
Make sure orange cloud is enabled for records you want to proxy through Cloudflare.


✅ 4. Update Nameservers

Cloudflare provides two nameservers.
Go to your domain registrar (e.g., GoDaddy, Namecheap).
Replace existing nameservers with Cloudflare’s nameservers.
